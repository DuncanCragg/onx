
ONX
---

An OS for ESP32-P4 boards and devices - especially the M5Stack Tab5.

An "OS" as in both "kernel" and "shell" (no FreeRTOS):

 - the "kernel" part provides an API for data object state handling and update propagation

 - the "shell" part is an HD/WXGA screen for exploring and updating those data objects

The first demo I'm working on is a smartphone-like device built on the Tab5 with an LTE
module, where the data objects are SMS messages and contacts and collections of those.
